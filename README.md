# Scallop
干货集中营Android app（MVP + RxJava2 + Dagger2 + Retrofit） 

## Features
* 功能完整，提供了API接口的基本都实现了，已实现的如下：
	* 干货列表
	* 闲读
	* 图片浏览
	* 设置
	* 搜索
* 结构清晰
* 支持繁体中文、简体中文
